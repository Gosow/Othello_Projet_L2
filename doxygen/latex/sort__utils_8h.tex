{\footnotesize\ttfamily typedef int($\ast$ sort\+\_\+utils\+\_\+compare\+\_\+function) (const void $\ast$const a, const void $\ast$const b)}



Compare function signature. 

The implementer has to ensure that when the value refernced by {\ttfamily a} is equal to the one addressed by {\ttfamily b}, than zero is returned. When the value identified by {\ttfamily a} is greater than the one pointed by {\ttfamily b}, than the return value has to be a positive integer, and otherwise a negative one.

Pseudocode\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{\textcolor{keywordflow}{if} (*a > *b) \textcolor{keywordflow}{return} +1;}
\DoxyCodeLine{\textcolor{keywordflow}{else} \textcolor{keywordflow}{if} (*a == *b) \textcolor{keywordflow}{return} 0;}
\DoxyCodeLine{\textcolor{keywordflow}{else} \textcolor{keywordflow}{return} -1;}
\end{DoxyCode}
